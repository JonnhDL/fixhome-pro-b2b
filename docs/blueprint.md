# **App Name**: FixHome Pro B2B

## Core Features:

- User Authentication: Secure user login and registration with role-based access control (Empresa, Profissional, Cliente).
- Admin Dashboard: Dashboard to manage the entire platform, create and publish projects (RFQs), evaluate proposals, and manage users.
- Client Dashboard: Dashboard to submit new project requirements, track their project's status, and review shortlisted contractor proposals.
- Contractor Dashboard: Dashboard to browse available projects/RFQs, express interest, and submit detailed technical/financial proposals.
- Project RFQ Workflow: Admins post detailed projects with technical documents, deadlines, and budgets. Verified contractors can view projects and submit their bids/proposals directly on the platform. Admins have a system to review, compare, and score the proposals to select a winner.
- Document Management: Implement a system using Firebase Storage for uploading and downloading project-related documents (e.g., blueprints, technical specifications, legal documents).
- Complete i18n Implementation: Portuguese translation using pt-PT.

## Style Guidelines:

- Primary color: Deep blue (#1A237E), conveying trust and professionalism, reminiscent of established B2B platforms.
- Background color: Light gray (#F5F5F5), creating a clean and modern look.
- Accent color: Muted teal (#4DB6AC), providing subtle visual interest.
- Headline font: 'Poppins', a geometric sans-serif font, for titles. Body font: 'PT Sans', a humanist sans-serif font, for descriptions.
- Clean and modern icons that are appropriate to the B2B space. Color to match accent color.
- Clear and intuitive layouts with a focus on usability. Responsive design for all devices.
- Subtle animations and transitions to improve user experience. Animation color should match accent color.
